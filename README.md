# HabitPulse

A beautiful daily habit tracker for Android with Material You (Material Design 3) design.

Build positive habits, track your streaks, and visualize your progress with a clean, modern interface that adapts to your device's color scheme.

## Features

- **Daily Habit Tracking** — Mark habits as done/undone each day with a simple tap
- **Streak Tracking** — See current streak and best streak for each habit
- **Categories** — Organize habits into Health, Fitness, Learning, Productivity, and Mindfulness
- **Statistics** — Weekly completion chart and per-habit stats
- **Material You Design** — Dynamic colors that match your wallpaper (Android 12+)
- **Dark Mode** — Full dark theme support
- **Onboarding** — Starts with 3 preset habits to get you going

## Tech Stack

- **Language:** Kotlin
- **UI:** Jetpack Compose with Material Design 3
- **Architecture:** MVVM (ViewModel + Repository)
- **Database:** Room
- **Preferences:** DataStore
- **Navigation:** Navigation Compose
- **Min SDK:** 26 (Android 8.0)
- **Target SDK:** 35 (Android 15)

## Build

1. Clone the repository
2. Open in Android Studio (Hedgehog or newer)
3. Sync Gradle
4. Run on a device or emulator (API 26+)

```bash
./gradlew assembleDebug
```
